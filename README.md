## Chapter 1: Strings, Numbers, and Math

---

### Problem Statement : Counting Duplicate Characters

Write a program that counts duplicate characters from a given `String`.

### Sample Input 1

```
"aabcc"
```

### Sample Output 1

```
{a=2, b=1, c=2}
```
